# Créditos

## Agradecimientos

Este proyecto no existiría sin el trabajo de la comunidad open source que construyó las herramientas sobre las que se sostiene:

- **OpenCode** — por la interacción que permitió dar forma y auditar este proyecto
- **Tauri v2** — por el puente nativo entre web y escritorio
- **React 19** — por el modelo de componentes que hizo posible el editor
- **Vite** — por el empaquetado ultrarrápido
- **TypeScript** — por la seguridad de tipos
- **Tailwind CSS** — por el utilitario que agilizó cada píxel
- **Zustand** — por el estado global sin ceremonia
- **TipTap / ProseMirror** — por el editor de texto enriquecido
- **Google Fonts** — por Space Grotesk y JetBrains Mono
- **Lucide React** — por los iconos
- **dnd-kit** — por el arrastrar y soltar
- **JSZip** — por la exportación comprimida
- **DOMPurify** — por la sanitización HTML
- **Rust y su ecosistema** — por el backend de Tauri

## El equipo

Tres personas, sin Llama hoy, empujando código real.

## Sobre este proyecto

Constructor-Web se construyó **sin IA por primera vez** (el editor visual, los 17 bloques, la exportación, el menú clásico), pero **con IA hasta hoy** (las sesiones de auditoría, documentación, y esta página de créditos). Ambas formas de trabajar son válidas. El resultado es lo que importa.

Si quieres continuar este proyecto a tu manera, con IA o sin ella, el código es tuyo. Es MIT. Adelante.

— *Constructor Web Contributors, 2026*
