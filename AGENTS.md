# AGENTS.md

## Repository Overview

Single-page application: `Constructor-Web.html` - A code-free web page editor.

## How to Investigate

**The executable source is the HTML file itself.**

1. **Open it in a browser** to test the editor interface directly
2. **Explore the HTML structure** - it contains all JavaScript, CSS, and markup
3. **No build/packaging required** - it works immediately
4. **Click the "Añadir" button** to open the component library and see available section types

## Key Information

### How to Test
- Open `Constructor-Web.html` in any modern browser
- Use the toolbar buttons to add/edit content
- Click "Vista Previa" to see rendered output
- Click "Exportar" to download HTML

### Core Capabilities
- Creates web pages without coding
- Supports hero, text, text-image, tabs, panels, cards, CTA, and spacer sections
- Drag-and-drop for reordering and resizing
- LocalStorage persistence (changes auto-save)
- Preview mode available
- Export to standalone HTML file

### Browser Features
- Contenteditable elements allow direct text editing
- Context menu with formatting (bold, italic, underline, links, images)
- Responsive design with mobile controls

### Usage Flow
1. `Constructor-Web.html` (opens immediately in browser)
2. Add sections from component library
3. Click sections to edit text/style
4. Click preview button to view results
5. Export to download standalone page

This is a standalone, interactive demo and production-ready application. No setup required.
